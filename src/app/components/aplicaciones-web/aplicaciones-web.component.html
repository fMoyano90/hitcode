¿Que es una aplicación web?

Son aplicaciones multiplataforma que se ejecutan mediante un navegador, lo cual permite que se ejecuten en casi todos los dispositivos modernos como computadores, celulares, tablets, televisores, relojes modernos, entre otros.
Se puede programar cualquier tipo de aplicación, desde un simple programa para organizar tareas hasta programas complejos como un sistema contable completo o una plataforma para lograr el teletrabajo en tu empresa. La imaginación es el límite.

Ejemplos

Existen muchas aplicaciones web que usamos día a día, como Facebook, Amazon, Netflix, Gmail, Google, YouTube, Google Maps, entre muchas otras. Se puede crear prácticamente cualquier tipo de plataforma o sistema con las funcionalidades que se deseen. Manejar, recopilar, respaldar y centralizar información, automatizar procesos, crear canales de comunicación, montar plataformas de teletrabajo o de enseñanza, brindar soporte y mucho más se hace posible con una aplicación web.

Sistemas Online e Intranet

Las aplicaciones web permiten ser usadas en dos entornos principalmente:

  1. En una intranet o red local, como en un grupo de oficina, donde la información se respaldará en los mismos computadores del grupo de trabajo sin requerir conexión a internet.
  2. En internet, donde todos se podrán conectar y la información se almacenará y centralizará en un servidor de la nube.
